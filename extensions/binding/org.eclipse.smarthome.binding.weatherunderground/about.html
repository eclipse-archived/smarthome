Weather Underground API

The Weather Underground API is an Internet service offered by The Weather Underground, LLC (WUL) that provides weather information for any location worldwide. This service is free of charge but there is a daily limit and minute rate limit to the number of requests that can be made to the API for free. It requires attribution, has rate limits and their Terms of Use have to be accepted.
