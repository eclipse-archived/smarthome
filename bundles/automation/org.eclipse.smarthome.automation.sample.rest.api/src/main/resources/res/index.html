SmartHome Automation Rest API Demo


This simple Automation REST client demonstrates the basic usage the Automation RESTful web service.

The HTML pages for the Automation REST Resources are : RULES , MODULE TYPES , TEMPLATES

These pages demonstrate the basic functionality of the Automation Rule Resource, Module Type Resource and Template Resource, provided by the Automation RESTful web service conforming to the common behaviors and definitions for RESTful web services.

They also demonstrate some of the auxiliaries which ease the implementation of error handling, resource pagination and resource representation customization.
